# 11. Partner Onboarding

## Journey
1. receive SDK and docs
2. integrate sale/refund/status
3. test with emulator / dongle
4. complete diagnostics checklist
5. pilot with real terminal
6. go live

## Path provides
- getting started guide
- API reference
- sample app
- scenarios
- issue checklist
- support bundle instructions
- compatibility matrix
- MCP Tools for the partners AI to utilise 

## Partner provides
- release process for SDK upgrades
- support contact
- test environment
- ability to export support bundles

## Mandatory readiness checks
- discovery works
- sale works
- refund works
- receipt data works
- support bundle export works
- version screen is present
- guided health check runs
- MCP Service Available
